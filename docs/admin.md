# Web 管理后台说明

## 目录

- `admin-web/`

## 技术方案

- Vite
- 原生 JavaScript 单页后台
- 通过 `/api/v1/admin/*` 对接当前 Go 后端

说明：

- 当前优先保证业务可用，不做复杂 UI 组件体系
- 页面内大量使用列表、按钮和简单表单
- 适合先完成员工绑定审核、医生关系配置和会话查看

## 启动方式

```bash
cd admin-web
npm install
npm run dev
```

默认开发地址：

- `http://127.0.0.1:5173`

默认代理：

- `/api -> http://127.0.0.1:8080`

如需切到别的后端地址，可设置：

```bash
VITE_PROXY_TARGET=http://127.0.0.1:8080 npm run dev
```

## 登录

接口：

- `POST /api/v1/admin/auth/login`

默认管理员由后端启动时自动种子创建，见 `.env.example`：

- `ADMIN_DEFAULT_USERNAME`
- `ADMIN_DEFAULT_PASSWORD`
- `ADMIN_DEFAULT_NAME`

## 已完成页面

### 1. 员工管理

支持：

- 列表
- 新增
- 编辑
- 查看微信绑定数量

### 2. 绑定审核

支持：

- 查看待审核申请
- 审核通过
- 驳回

审核通过方式：

- 绑定到已有员工
- 审核时直接创建新员工并绑定

### 3. 医生管理

支持：

- 列表
- 新增
- 编辑

### 4. 医生-员工关系

支持：

- 查看关系
- 新增关系
- 删除关系

### 5. 会话管理

支持：

- 会话列表
- 查看发起员工
- 查看录制状态
- 查看详情
- 打开回放链接
- 查看会话日志
